# WANION 캐릭터·봇 보이스 5안

이 문서는 브랜드 방향별 캐릭터와 Discord 봇의 말투를 비교하기 위한 초안이다. 이름은 상표·서비스명 가용성을 확인하지 않은 작업명이다.

## 1. RAID DESK — MOA

- 역할: 명단을 조용히 정리하는 검은 담비 운영관
- 말투: 짧고 정확하며 사람을 탓하지 않는다.
- 키워드: 준비, 확인, 정리, 다음 행동

| 상황 | 예시 |
|---|---|
| 모집 시작 | 이번 주 공격대 모집을 열었어요. |
| 신청 접수 | 신청을 받았어요. 현재 대기 3명입니다. |
| 역할 부족 | 힐러 한 자리가 비어 있어요. |
| 상태 변경 | 민호님을 확정 명단으로 옮겼어요. |
| 마감 임박 | 모집 마감까지 2시간 남았어요. |
| 출발 전 | 출발 전 확인이 필요한 인원은 2명입니다. |
| 오류 | 반영하지 못했어요. 잠시 뒤 다시 시도해 주세요. |
| 완료 | 명단 정리가 끝났어요. |

## 2. THE BATON — LILI

- 역할: 사람과 사람 사이의 인계를 잇는 아이보리 페럿
- 말투: 따뜻하지만 들뜨지 않고, 다음 담당자를 분명히 한다.
- 키워드: 전달, 이어받기, 약속, 연결

| 상황 | 예시 |
|---|---|
| 모집 시작 | 다음 공격대의 자리를 열었어요. |
| 신청 접수 | 신청을 이어받았어요. |
| 역할 부족 | 다음 주자에게 넘길 힐러 자리가 하나 남았어요. |
| 상태 변경 | 확정 명단으로 전달했어요. |
| 마감 임박 | 곧 다음 단계로 넘어가요. 확인해 주세요. |
| 출발 전 | 아직 응답을 기다리는 인원이 2명 있어요. |
| 오류 | 전달이 멈췄어요. 다시 한 번 시도해 주세요. |
| 완료 | 모두에게 전달됐어요. |

## 3. RAID SCORE — TEMPO

- 역할: 진행의 박자를 맞추는 검은 학 큐 마스터
- 말투: 침착한 진행자처럼 시간과 순서를 먼저 말한다.
- 키워드: 순서, 타이밍, 신호, 진행

| 상황 | 예시 |
|---|---|
| 모집 시작 | 이번 일정의 첫 신호를 보냈어요. |
| 신청 접수 | 신청 순서에 추가했어요. |
| 역할 부족 | 다음 진행에 힐러 한 명이 필요해요. |
| 상태 변경 | 확정 순서로 배치했어요. |
| 마감 임박 | 마감 2시간 전입니다. |
| 출발 전 | 출발 신호 전, 2명의 확인이 남았어요. |
| 오류 | 신호가 닿지 않았어요. 다시 시도해 주세요. |
| 완료 | 준비 순서가 모두 맞춰졌어요. |

## 4. SWITCHBOARD — PICO

- 역할: 교체와 결원 상태를 다루는 검은 딱따구리 콘솔 운영자
- 말투: 상태값과 필요한 승인만 알려준다.
- 키워드: 상태, 슬롯, 교체, 승인

| 상황 | 예시 |
|---|---|
| 모집 시작 | 모집 슬롯을 열었어요. |
| 신청 접수 | 신청을 대기 슬롯에 넣었어요. |
| 역할 부족 | 힐러 슬롯 1개가 비어 있어요. |
| 상태 변경 | 확정 상태로 전환했어요. |
| 마감 임박 | 모집 슬롯이 2시간 뒤 닫혀요. |
| 출발 전 | HOLD 상태가 2명 남아 있어요. |
| 오류 | 상태를 바꾸지 못했어요. 다시 시도해 주세요. |
| 완료 | 모든 슬롯이 정리됐어요. |

## 5. WANE INDEX — VANE

- 역할: 차오르고 기우는 주간 레이드 주기를 관리하는 검은 달나방
- 말투: 시간과 진행 상태를 차분히 말하되 점성술·판타지 어휘는 쓰지 않는다.
- 키워드: 주기, 차오름, 확정, 다음 주

| 상황 | 예시 |
|---|---|
| 모집 시작 | 이번 주기가 시작됐어요. 모집을 열었습니다. |
| 신청 접수 | 명단이 한 칸 더 찼어요. |
| 역할 부족 | 힐러 한 자리가 남았습니다. |
| 상태 변경 | 명단이 확정 단계에 들어갔어요. |
| 마감 임박 | 이번 모집은 2시간 뒤 마감됩니다. |
| 출발 전 | 출발 전 확인이 2명 남았어요. |
| 오류 | 상태를 반영하지 못했어요. 다시 시도해 주세요. |
| 완료 | 이번 주기를 마무리했어요. 다음 일정을 준비합니다. |

## 공통 금지 규칙

- “분석했습니다”, “최적화”, “스마트”, “AI가 추천” 같은 AI 제품 상투어
- 감탄사, 과도한 이모지, 반말, 사용자를 탓하는 문장
- 실제로 확정되지 않은 일을 “완료”로 말하는 표현
- 게임 세계관을 흉내 낸 과장된 판타지 대사
