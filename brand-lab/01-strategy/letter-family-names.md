# WANION Letter Family Names

## 원칙

엄격한 완전 애너그램 하나를 억지로 이름으로 만들기보다 `W A N I O N`에 들어 있는 글자만 사용해 같은 가족처럼 들리는 이름 체계를 만든다. 새 알파벳을 추가하지 않으며, 캐릭터 이름 둘을 나란히 놓았을 때 WANION이 자연스럽게 연상되어야 한다.

## 권고 구조

- **WANI / 와니** — 대표 캐릭터. 브랜드의 얼굴이자 사용자가 부르는 친근한 이름.
- **NION / 니온** — Discord·시스템 봇. 상태와 주기를 안내하는 조금 더 정밀한 인상.
- 결합: `WANI + NION → WANION`

와니는 사람과 커뮤니티의 온도를, 니온은 운영 시스템의 정확성을 담당한다. 두 캐릭터를 분리하면 마스코트와 봇의 역할이 겹치지 않는다.

## 최종 대상

| 이름 | 동물 | 역할 |
|---|---|---|
| WANI / 와니 | 2번 아이보리 암컷 족제비 | 메인 캐릭터. 모집·정렬·준비를 돕는 운영관 |
| NION / 니온 | 6번 검은 까마귀 | Discord 봇. 끝난 주기의 실패·불참·미응답을 기록하는 curse keeper |

다른 애너그램 이름과 다른 동물 캐릭터는 사용하지 않는다. 5개 브랜드 방향은 이 두 캐릭터가 입는 시각 시스템을 비교하는 안이다.

## 검증 전제

이름 후보는 언어적 1차 검토일 뿐이다. 캐릭터·봇 이름으로 외부 공개하기 전 상표, Discord 봇, 소셜 핸들, 도메인 및 주요 언어권의 부정 의미를 확인해야 한다.
