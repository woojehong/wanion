# 디자인 베타 수용 체크리스트

## 기능 보존

- [ ] 신청 자격 3단계 유지
- [ ] 역할 정원과 자동 대기 유지
- [ ] 대기·벤치·승격·강등 유지
- [ ] 스왑 가능자 표시 유지
- [ ] 시너지 트래커 유지
- [ ] 파티 배치 시뮬레이션 유지
- [ ] 초대·배치 코드 유지
- [ ] 예약·메모·취소 기록 유지
- [ ] 권한별 화면과 동작 유지
- [ ] 보드/달력/내 일정 3뷰 유지

## 시각

- [ ] `#8A70FF`, `#09090D` 핵심색 유지
- [ ] 보라 그라디언트 없음
- [ ] 과한 글로우·유리·포털·회로·별자리 없음
- [ ] 한 화면 Primary CTA 1개 원칙
- [ ] 정보 위계가 색상 없이도 읽힘
- [ ] WANI/NION 외 캐릭터 없음
- [ ] 캐릭터가 데이터 화면을 방해하지 않음
- [ ] `RUN BY STUDIO HOOJE`가 WANION보다 작음

## 반응형

- [ ] 360px
- [ ] 390px
- [ ] 768px
- [ ] 1024px
- [ ] 1440px
- [ ] 가로 스크롤은 의도한 테이블/스트립에만 존재
- [ ] 모바일 핵심 CTA가 접근 가능
- [ ] 달력 복수 일정이 겹치지 않음

## 접근성

- [ ] 키보드 포커스 표시
- [ ] 버튼/링크 의미가 명확
- [ ] 색상 외 상태 텍스트
- [ ] 본문 대비 WCAG AA 목표
- [ ] 입력 라벨과 오류 연결
- [ ] 아이콘에 accessible name
- [ ] 모달 포커스와 닫기 동작

## 상태 검증

- [ ] 로딩
- [ ] 빈 상태
- [ ] 오류
- [ ] 권한 없음
- [ ] 미로그인
- [ ] BNet 미연동
- [ ] 모집 중
- [ ] 마감
- [ ] 대기
- [ ] 벤치
- [ ] 손님파티
- [ ] 복수 파티
- [ ] 일정 충돌

## 실행 검증

- [ ] `npm run build`
- [ ] 린트/테스트가 있으면 실행
- [ ] 라우트별 수동 확인
- [ ] 콘솔 오류 없음
- [ ] 기존 데이터와 빈 데이터 모두 확인
- [ ] 변경 파일 외 예상치 못한 수정 없음

## 실패 조건

- 핵심 기능 하나라도 사라짐
- 모바일에서 신청 또는 운영 기능 접근 불가
- 기존 데이터 계약을 디자인 편의 때문에 변경
- 샘플 데이터를 운영 데이터처럼 연결
- 캐릭터를 3개 이상 사용
- RallyPoint 또는 WoW 공식 시각물과 혼동될 정도의 모방
