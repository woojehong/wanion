# RallyPoint 참고와 독창성 경계

## 흡수할 강점

- 로그인 전 파티 탐색
- 검색→필터→카드→신청의 짧은 흐름
- 카드 안에서 난이도·역할·시간·인원·리더를 빠르게 확인
- BNet 캐릭터 정보가 신청 흐름으로 이어지는 구조
- 생성·알림·프로필 액션의 명확한 위치

이는 특정 서비스의 소유물이 아닌 일반적인 UX 원칙 수준에서만 참고한다.

## 복제 금지

- 청록색 전술 콘솔 팔레트
- 레이더·격자·조준선·코너 브래킷
- Tactical 계열 명명과 마이크로 카피
- 히어로 문장, 검색창, 카드의 동일한 배치 순서
- 아이콘과 장식 패턴
- 공식 WoW 키아트의 유사 사용

## WANION만의 차별점

- 길드·공격대·연합이 분리된 실제 조직 구조
- 복수 소속과 반복 운영
- 시너지·스왑·대기·벤치·배치 시뮬레이션
- 주간 운영 주기를 표현하는 WANE INDEX
- 로스터 바와 기준선을 사용하는 RAID DESK
- WANI/NION 두 캐릭터의 역할 분리

## 병렬 비교 기준

두 화면을 나란히 놓았을 때 아래가 모두 달라야 한다.

- 주색과 보조색
- 페이지 셸
- 필터 배치
- 카드 해부
- 상태 표현
- 타이포 위계
- 그래픽 모티프
- 카피와 버튼 문구
