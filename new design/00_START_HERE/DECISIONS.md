# 현재 확정사항

## 제품

- 기존 한길련 시스템의 장점과 핵심 운영 로직 유지
- RallyPoint의 발견성·필터·일정 가독성은 흡수하되 모방하지 않음
- 파티 보드와 달력 전환
- 하루 여러 파티 표시
- 길드·공격대·연합·무소속 공대장 구조 유지

## 디자인

- 모던·세련·프로페셔널
- AI 생성 서비스처럼 보이지 않음
- RAID DESK를 주 시각 문법으로 사용
- WANE INDEX를 주간 상태 언어로 사용
- SWITCHBOARD의 결원·상태 탭을 보조 사용
- 핵심색 `#8A70FF`, 배경 `#09090D`
- `RUN BY STUDIO HOOJE`

## 캐릭터

- WANI: 암컷 아이보리 족제비
- NION: 조금 성숙한 검은 새 봇
- 두 캐릭터만 사용
- `WANI + NION = WANION`

## 스토리

- 레이드는 달과 같다.
- 우리의 운영 주기는 일주일이다.
- 모집으로 차오르고, 확정으로 만월에 닿고, 출발과 기록을 지나 기운 뒤 다음 주기를 준비한다.
- 저주는 반복되는 나쁜 운영 패턴의 은유이며 공포·오컬트로 표현하지 않는다.

## 아직 확정하지 않음

- 최종 로고 심벌
- 최종 워드마크 폰트와 자간
- WANI/NION의 투명 배경 벡터 원본
- 포인트 보상 수치
- 상표·도메인·봇 이름 가용성
