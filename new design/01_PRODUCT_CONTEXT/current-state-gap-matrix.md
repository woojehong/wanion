## 작업 전 확인할 핵심

1. `npm run build`
2. 모든 라우트 접근
3. 신청/취소/승인/대기/벤치/스왑 액션
4. 시뮬레이터와 초대 코드
5. 360px 모바일
6. 빈 데이터와 복수 일정
