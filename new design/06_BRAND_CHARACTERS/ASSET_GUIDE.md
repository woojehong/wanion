# WANI & NION Asset Guide

## 활성 캐릭터

### `WANI_canonical.png`

- 유일한 메인 캐릭터
- 아이보리 암컷 족제비
- 도회적이고 세련된 운영관
- 모집·정렬·준비·온보딩·일반 빈 상태

### `NION_canonical.png`

- 유일한 봇 캐릭터
- 뾰족한 검은 머리, 흰 얼굴 패치, 보라색 탭을 가진 검은 새
- Discord·기록·회고·미응답·불참 정리

## 포즈 아카이브

- `archive_WANI_pose-sheet.png`
- `archive_NION_pose-sheet.png`

포즈 아카이브에 같은 캐릭터가 여러 번 그려져 있어도 여러 캐릭터가 아니다. 새 동물이나 새 이름을 만들지 않는다.

## 사용 제한

- 제품 화면에 허용되는 이름은 WANI, NION뿐
- 한 화면 최대 한 캐릭터
- 데이터 카드와 로스터 행에 반복하지 않음
- WANI를 선정적으로 의인화하지 않음
- NION을 붉은 눈·주술·고딕·공포 캐릭터로 만들지 않음
- 최종 제품용 투명 배경 PNG·SVG는 아직 확정 자산이 아니다. 현재 이미지는 방향 참고용이다.

## 이미지 생성 기록

현재 캐릭터는 내부 이미지 생성 도구로 제작된 콘셉트 시안이다. 최종 상업용 자산으로 확정할 때는:

1. 동일 실루엣의 수작업 벡터 재작도
2. 투명 배경
3. 32/64/128/512px 축소 검증
4. Discord 원형 크롭
5. 상표·유사 캐릭터 검토
6. 프롬프트와 후편집 내역 보존
